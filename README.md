# rozzoponte-website
Rozzo Ponte Italian restaurant website
イタリアンレストランの公式ウェブサイト制作プロジェクト。

# 公開サイト
https://rozzoponte.jp/

# 使用技術
- **HTML5 / CSS3**
- **JavaScript (Vanilla JS)**
- Responsive Design (モバイルファースト)
- Google Fonts
- Formspree / Google Maps など

# 制作内容
- ホームページ
- メニュー紹介
- ワイン紹介
- ギャラリー
- 店舗情報・アクセス（Google Maps埋め込み）
- お問い合わせなど

# 工夫した点
- レスポンシブデザイン（スマホ対応）
- レストランの温かみと上品さを表現した落ち着いたデザイン
- 写真を活かしたビジュアル
- 電話予約の可能性
- シンプルでメンテナンスしやすい構造

# 学んだこと
- クライアント（店舗オーナー）の意向を形にするコミュニケーション
- 実際のビジネスのウェブサイトに求められる要件
- デザインと機能性のバランスの取り方
